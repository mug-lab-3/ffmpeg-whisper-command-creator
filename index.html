Whisper Command Creator

Generate FFmpeg commands for Whisper functionality effortlessly.

How to get the full path: ▼
  • Hold Shift and Right-Click the file, then select "Copy as path".
  • Right-Click the file, hold Option (Alt), then select "Copy 'filename' as Pathname".
Setup Required ▼
Create a models folder relative to your FFmpeg: 
Your-FFmpeg-Root/
├── bin/
│   └── ffmpeg.exe  <-- Run this
└── models/
    ├── ggml-large-v3.bin
    └── ...
Create a models folder in the same folder as ffmpeg: 
Your-Folder/
├── ffmpeg       <-- Run this
└── models/
    ├── ggml-large-v3.bin
    └── ...
